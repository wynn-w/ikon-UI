# 输入框 - Input

通过鼠标或键盘输入内容

# 基础用法

<template>
  <div class="sources">
    <ik-row>
      <ik-input v-model="input1" placeholder="请输入内容"></ik-input>
    </ik-row>
  </div>
</template>
<script>
export default {
  name: "app",
  data() {
    return {
      input1: "",
    };
  },
};
</script>

# 禁用/只读状态

通过 disabled/readonly 属性指定是否禁用/只读 input 组件

<template>
  <div class="sources">
    <ik-row>
      <ik-input v-model="input2" placeholder="禁用状态" disabled></ik-input>
      <ik-input v-model="input3" placeholder="只读状态" readonly></ik-input>
    </ik-row>
  </div>
</template>
<script>
export default {
  name: "app",
  data() {
    return {
      input2: "",
      input3: "",
    };
  },
};
</script>
<style>
.ik-row {
  margin-top: 10px;
}
.ik-row > .ik-input {
  margin: 4px 10px 0;
  width: 140px;
}
</style>

# 密码框

使用 showPassword 属性即可得到一个可切换显示隐藏的密码框

<template>
  <div class="sources">
    <ik-row>
      <ik-input
        v-model="input5"
        type="password"
        placeholder="普通密码框"
      ></ik-input>
      <ik-input
        v-model="input6"
        placeholder="可切换密码框"
        showPassword
      ></ik-input>
    </ik-row>
  </div>
</template>
<script>
export default {
  name: "app",
  data() {
    return {
      input5: "",
      input6: "",
    };
  },
};
</script>
<style>
.ik-row {
  margin-top: 10px;
}
.ik-row > .ik-input {
  margin: 4px 10px 0;
  width: 140px;
}
</style>

# 尺寸

可通过 size 属性指定输入框的尺寸，除了默认的大小外，还提供了 medium、small 和 mini 三种尺寸。

<template>
  <div class="sources">
    <ik-row class="diff">
      <ik-input v-model="input7" placeholder="默认输入框"></ik-input>
      <ik-input
        v-model="input8"
        placeholder="中等输入框"
        size="medium"
      ></ik-input>
      <ik-input
        v-model="input9"
        placeholder="小型输入框"
        size="small"
      ></ik-input>
      <ik-input
        v-model="input10"
        placeholder="迷你输入框"
        size="mini"
      ></ik-input>
    </ik-row>
  </div>
</template>
<script>
export default {
  name: "app",
  data() {
    return {
      input7: "",
      input8: "",
      input9: "",
      input10: "",
    };
  },
};
</script>
<style>
.ik-row {
  margin-top: 10px;
}
.ik-row > .ik-input {
  margin: 4px 10px 0;
  width: 140px;
}
</style>

# Attributes

属性名 类型 默认值 可选值 说明
type String number/text/password 类型
value String/Number 绑定值
disabled Boolean false 页设置显示图标
readonly Boolean false 图标位置
clearable Boolean false 是否可清空
showPassword Boolean false 是否显示切换密码图标
size String text mini/small/medium 尺寸
id String 原生属性，id
name String 原生属性，name
placeholder String 原生属性，输入框占位文本
step Number 原生属性，步长
maxlength Number 原生属性，最大输入长度
min Number 原生属性，最小输入值
max Number 原生属性，最大输入值
